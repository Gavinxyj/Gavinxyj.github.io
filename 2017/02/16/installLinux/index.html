安装步骤

这里以Centos6.5为例

U盘启动

将U盘查到电脑上，重启电脑，在开机引导界面按下F12,我的主机是ThinkPad的，其它品牌的机器可能有所区别，按下F12后在出现的界面上选择以USB启动，稍等片刻出现以下画面

界面说明：

  • Install or upgrade an existing system 安装和升级现有系统
  • Install system with basic video driver 安装过程中采用基本的显卡驱动
  • Rescue installed system 进入系统修复模式
  • Boot from local drive 退出安装从硬盘启动
  • Memory test 内存检测

我们这里选择第一项，然后按回车,出现一下界面

选择English后出现下图所示

keyboard Type键盘类型选择us后出现下图所示

选择Hard drive硬件驱动后出现下图所示

这里不同的机器有可能出现不同的选项，有可能会出现/dev/sda1;/dev/sda2等等，这里你只需要选择你的镜像文件在哪个磁盘位置就可以了，我这里选择/dev/sda4这就是我的U盘，回车后出现下图所示，然后点击next

语言选择，键盘选择

语言选择可以选择简体中文，和English，最好是选择English，选择好后点击“next”出现下图所示

键盘选择美国英语式，点击“next”

选择安装设备


选择“基本存储设备”，点击下一步出现下图所示，如果你不是在虚拟机上装的可能不会出现下图，所以如果你在真机上安装没有出现下图，请不必慌张。

选择“是，忽略所有数据”

设置主机名


这里取个主机名字，点击下一步

设置时区


时区设置为亚洲上海，点击下一步

设置root密码


设置root密码，然后点击“下一步”，如果你的密码过于简单会出现下图所示

选择“无论如何都使用”

设置分区


现在“创建自定义布局”，单机下一步

注意：创建分区要遵守如下原则

  • 必须创建 / 分区，大小为50-100GB
  • 必须创建 swap 分区，大小为内存2倍
  • 必须创建 /boot分区，大小为200M
  • 必须创建 /app 分区，大小为200GB

鼠标点中sda,然后点击创建，出现下图所示

选择“标准分区”，然后点击“创建”

这里我的根目录分配的10G是因为我使用虚拟机安装的，若在真机上按照此根目录大小可以根据实际情况制定。创建“/”分区，然后点击确定，出现下图所示

点击“创建”，创建下一个分区/boot,分区大小为200MB

选择“标准分区”，点击“创建”

点击“确定”，/boot分区创建完成，出现下图所示

点击“创建”，创建下一个分区swap,分区大小为1024MB

选择“标准分区”，然后点击“创建”

选择“文件系统类型为swap”，大小内存2倍。点击“确定”

点击“创建”，创建/app 分区

选择“标准分区”，点击“创建”

挂载点输入/app,大小为磁盘剩余空间，点击“确定”，分区分配完成。

点击“下一步”,出现如下图所示

点击“将修改写入磁盘”

安装引导程序

这一步非常，非常重要，因为我们是用U盘按照的，所以这里可能会出现“在/dev/sdbx中安装引导程序”，如果你看见的是这个的话一定要选择更换设备，在弹出来的对话框选择将引导程序安装在/dev/sda上，切记，切记！！！。

选择好后，单机“下一步”

选择安装包

这里你可以选择基本服务，基本服务就是没有界面终端的，如果你想要有像windows桌面一样的服务你就选择桌面，这里我选择的是基本服务

选择“现在自定义”，点击“下一步”出现下图所示

等待安装完成出现下图，点击重新引导完成系统安装

系统重启后，以管理员身份登陆，在命令行下执行命令后，修改网卡配置如下图所示


修改网卡配置文件，修改参数 onboot=yes，保存退出（Esc + wq），重启网卡

关闭防火墙，设置防火墙开机不启动。
